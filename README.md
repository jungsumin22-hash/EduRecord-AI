# EduRecord AI Pro v30 DB

구성:
- index.html: DB 기반 행특 생성 엔진 초안
- data/personality.json: 성격 표현 DB
- data/behaviors.json: 구체 행동 DB
- data/teacher_phrases.json: 교사 문체 표현 DB
- data/growth_map.json: 성장 포인트 변환 DB
- data/connectors.json: 연결어/어미 DB
- data/competency_map.json: 역량 매핑 DB
- data/forbidden.json: 위험 표현 대체 DB

기존 EduRecord-AI 폴더에 압축을 풀어 덮어쓰기 → Commit → Push
